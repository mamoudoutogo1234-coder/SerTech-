# Projet d'Automatisation de l'Agriculture et de l'Arrosage

## Description

Ce projet vise à **optimiser et automatiser le processus agricole** grâce à des capteurs et des systèmes d’irrigation intelligents.  
Il permet de surveiller l’humidité du sol, la température, et d’activer automatiquement l’arrosage pour améliorer le rendement et économiser l’eau.

---

## Fonctionnalités

-   Surveillance en temps réel :
    -   Humidité du sol
    -   Température et climat ambiant
-   Arrosage automatique selon les seuils configurés
-   Historique et suivi des données agricoles
-   Alertes et notifications (optionnelles)
-   Interface utilisateur simple pour visualiser l’état des cultures et contrôler l’irrigation

---

## Technologies Utilisées

-   **Hardware** : Arduino / Raspberry Pi, capteurs d’humidité, relais pour l’arrosage
-   **Logiciel** :
    -   Backend : Python / Node.js
    -   Frontend : Vue.js / React / Laravel (selon choix)
    -   Base de données : MySQL / SQLite
-   **Communication** : Wi-Fi / Bluetooth / MQTT
